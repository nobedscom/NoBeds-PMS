# NoBeds PMS iOS App (Hotel Property Management System)

**NoBeds PMS** is a lightweight iOS application built with **SwiftUI** that provides mobile access to the **NoBeds Property Management System (PMS)** via embedded WebView.

This app is designed for hotel owners, apartment managers, and short-term rental operators who need fast access to their **hotel dashboard**, **booking management**, **availability calendar**, and **channel manager** features directly from an iPhone or iPad.

The NoBeds platform helps automate daily property operations such as managing reservations, pricing, occupancy, and OTA integrations.
